Ändern der Audio Parameter

Tonhöhe

  <honey-bibsonomy id="pitch0"  textids="vorwort" title="pitch0 (Höhe 0.1)"
                audiopitch="0.1"></honey-bibsonomy>
  <honey-bibsonomy id="pitch1"  textids="vorwort" title="pitch1 (Höhe 1 = default)"
                audiopitch="1"></honey-bibsonomy>
  <honey-bibsonomy id="pitch2"  textids="vorwort" title="pitch2 (Höhe 2 :-) )"
                audiopitch="2"></honey-bibsonomy>

Styling der Komponente

Globale Definitionen um alle Komponenten identisch zu stylen

  /* <style> set all speaker at page to defaults of component via :root */
    :root {
      --honey-bibsonomy-background: transparent;
      --honey-bibsonomy-color: blue;
      --honey-bibsonomy-padding: 5px;
      --honey-bibsonomy-font-size: medium;
      --honey-bibsonomy-border: 0;
      --honey-bibsonomy-width: 36px;
      --honey-bibsonomy-height: 36px;
    }

Styling über css class und custom properties verändert

  # part of style sheet

    #css1 {
      --honey-bibsonomy-color: gray;
    }

    .css2class {
      --honey-bibsonomy-color: red;
      --honey-bibsonomy-background: #E8CD9A;
    }

    #css3 {
      --honey-bibsonomy-color: green;
    }
    .css3class {
      --honey-bibsonomy-background: #E8CD9A;
    }

  # part of html

    <honey-bibsonomy id="css1" textids="artikel2"></honey-bibsonomy>
    <honey-bibsonomy textids="artikel2" class="css2class"></honey-bibsonomy>
    <honey-bibsonomy id="css3" textids="artikel2" class="css3class"></honey-bibsonomy>
